# Vision

---

# Mission

Mengurangi pekerjaan administratif yang berulang melalui digitalisasi seluruh proses event.

Mulai dari:

* Registrasi
* Database Peserta
* Absensi
* Penilaian
* Dashboard
* Sertifikat
* Arsip

Dalam satu platform.

---

# Background

Awalnya sistem ini dibuat untuk membantu operasional kegiatan CAI.

Selama pengembangan ditemukan bahwa hampir semua event memiliki masalah yang sama:

* Data peserta dibuat ulang setiap tahun.
* Registrasi masih manual.
* QR dibuat menggunakan website pihak ketiga.
* Laporan dibuat menggunakan spreadsheet.
* Penilaian terpisah dari sistem.
* Arsip tersebar di berbagai tempat.

Dari kebutuhan tersebut lahirlah KJA Event Manager.

---

# Problem

Organisasi sering mengalami:

* Data peserta tidak berkelanjutan.
* Banyak pekerjaan manual.
* Kesalahan input data.
* Sulit mencari histori peserta.
* Sulit mengetahui progress event secara realtime.
* Banyak aplikasi terpisah untuk satu event.

---

---

# Product Principles

Setiap fitur baru harus memenuhi minimal satu tujuan berikut:

* Mengurangi pekerjaan manual.
* Mengurangi kemungkinan human error.
* Mempercepat operasional event.
* Mempermudah monitoring.
* Dapat digunakan kembali pada event berikutnya.
* Tidak menambah kompleksitas yang tidak diperlukan.

Jika sebuah fitur tidak memberikan manfaat nyata terhadap operasional event, maka fitur tersebut harus dipertimbangkan kembali sebelum dikembangkan.

---

# Final Vision

KJA Event Manager bukan sekadar aplikasi absensi.

KJA Event Manager adalah platform Event Management yang dibangun untuk membantu organisasi mengelola peserta, panitia, kompetisi, dan seluruh operasional event secara terintegrasi, dengan tujuan menjadi solusi yang dapat digunakan lintas organisasi dan berkembang menjadi produk komersial di masa depan.
